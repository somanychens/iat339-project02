System requirements

Can my PC run the game?

The minimum requirements to run the game on a PC are:

  • OS: Windows 7 or higher 64bit
  • Processor: Intel Core i3 2.00 GHz or AMD equivalent
  • Memory: 6GB RAM
  • Graphics: NVIDIA GeForce 450 or higher with 1GB Memory
  • Storage: 4GB available space
  • DirectX: DX11

What if I have a 32bit operating system?

32bit OS's are only able to utilize a maximum of ~3GB of system memory, and are unsupported at this time.

Known issues & fixes

I'm stuck and can't move my character!

We have implemented an "Unstuck" button in the Pause menu! It will teleport you to the nearest waypoint on the map.

Return to top

The game keeps crashing!

There are a few common causes for this issue:

  • You are running the game with out of date hardware or drivers that don't support full DX11 features on Windows 7, 8, or 10.
  • You are attempting to run the game with less than 6GB of RAM.
  • You are running the game with very little space (<2GB) available on your OS hard drive (typically C: on Windows).
  • Your anti-virus or similar program is for some reason protecting writing and reading from the game's save folders (on Windows this is in %APP_DATA% on your OS drive).

You should ensure you have installed the latest graphics drivers for your computer.

If you are still experiencing crashes, it may be helpful to send us your machine's specs and a log file that we can use to categorize your issue. Please attach this file to your support email when contacting us at support@thegame.com to speed up the process.

I used the wrong e-mail and never got my Serial Code or confirmation of my purchase!

Please submit a support request to support@thegame.com and provide the following information:

  • The incorrect e-mail address when you made the purchase (if you rememeber it)
  • The Zip code associated with your method of payment
  • Credit/Debit: LAST 4 DIGITS ONLY
  • PayPal:
    • The e-mail address associated with the PayPal account
    • Whether the PayPal account is verified or unverified
    • The country in which the PayPal account was created or set up
    • The Invoice ID from your PayPal payment history for the transaction

Can I get a refund?

We are happy to review your refund request. Please submit a request to support@thegame.com and provide the following information:

  • The e-mail address you used when making the purchase (if available)
  • Your billing zip code or postal code
  • Credit/Debit: LAST 4 DIGITS ONLY
  • PayPal: The invoice ID or transaction ID from your PayPal payment history for the transaction
