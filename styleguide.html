Interactive Elements

In-text link

CSS selector:

a

Sample code:

<p>Here is an example of an <a href="">in-text link</a> to nowhere.</p>

Rendered element:

Here is an example of an in-text link to nowhere.

Navigation button

CSS selector:

.button

Sample code:

<a href="" class="button">Navigation button</a>

Rendered element:

Navigation button

Submit and reset button

CSS selector:

.form-button

Sample code:

<button type="submit" value="Submit" class="form-button">Submit</button>
<button type="reset" value="Reset" class="form-button">Reset</button>

Rendered element:

Text input field and label

CSS selector:

.form-text

Sample code:

<form>
   <p>Search bar:</p>
   <input type="text" name="searchbar" class="form-text">
</form>

ul, li

Sample code:

   <li>List item 1</li>

</ul>

ol, li

.main-nav, .nav-left, .nav-middle, .nav-right, .grid-col-1of3, .button

Sample code:

<nav class="main-nav">
   <div class="grid-col-1of3 nav-left">
      <a href="index.html" class="button">Home</a>
   </div>
   <div class="grid-col-1of3 nav-middle">
      <a href="game.html" class="button">Game</a>
      <a href="news.html" class="button">News</a>
      <a href="store.html" class="button">Store</a>
   </div>
   <div class="grid-col-1of3 nav-right">
      <p class="button">Cart</p>
   </div>
</nav>
