Interactive Elements

In-text link

CSS selector:

a

Sample code:

<p>Here is an example of an <a href="">in-text link</a> to nowhere.</p>

Rendered element:

Here is an example of an in-text link to nowhere.

Navigation button

CSS selector:

.nav-button

Sample code:

<a href="" class="button">Navigation button</a>

Rendered element:

Navigation button

Submit and reset button

CSS selector:

.form-button

Sample code:

<button type="submit" value="Submit" class="form-button">Submit</button>
<button type="reset" value="Reset" class="form-button">Reset</button>

Rendered element:

Text input field and label

CSS selector:

.form-text

Sample code:

<form>
   <p>Search bar:</p>
   <input type="text" name="searchbar" class="form-text">
</form>

Rendered element:

Search bar:

Bulletted list

CSS selector:

ul, li

Sample code:

<ul>
   <li>List item 1</li>
   <li>List item 2</li>
   <li>List item 3</li>
</ul>

Rendered element:

  • List item 1
  • List item 2
  • List item 3

Numbered list

CSS selector:

ol, li

Sample code:

<ol>
   <li>List item 1</li>
   <li>List item 2</li>
   <li>List item 3</li>
</ol>

Rendered element:

  1. List item 1
  2. List item 2
  3. List item 3

Combined Elements

Main navigation

CSS selector:

.main-nav, .nav-home, .nav-button

Sample code:

<nav class="main-nav">
   <a href="" class="nav-home">Home</a>
   <a href="" class="button">Game</a>
   <a href="" class="button">News</a>
   <a href="" class="button">Store</a>
</nav>

Rendered element:

Home Game News Store

Product listing

CSS selector:

.product, .clear, .product-image, .product-content, .price, .buy-button

Sample code:

<div class="product clear">
   <img src="images/template-image.jpg" class="product-image" alt="Product image">
   <div class="product-content">
      <h4>Product</h4>
      <p>A paragraph describing what this product is. Here are some details about the product, such as what material it comes in. Here is a feature or highlight about the product that will make you want to buy it. Comes in multiple sizes!</p>
      <div class="price">
         <p><strong>$10 CAD</strong></p>
      </div>
      <div class="buy-button">
         <a href="" class="button buy-button">Buy</a>
      </div>
   </div>
</div>

Rendered element:

Product

   </nav>
